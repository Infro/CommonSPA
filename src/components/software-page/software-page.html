Custom Software Development

One of the last resorts in the toolbox. One of the most powerful tools, and with it the potential for doing great things as well as doing incredibly stupid things.

How do you know you if you need custom software development? Forbes has provided a nice article which contains within some great insight about choosing when and why. The short version being, use custom software when you're doing something that no one else has done, and the expected return on investment is positive.

Although full blown custom software development isn't for every job, in the right environment custom software developers shine for their ability to take other people's jobs and automate parts which are suitable for automation, and provide tools when calculations, visualizations, or other factors can augment a worker's productivity.
